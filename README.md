# 防盗门进销存管理系统（SDIMS）

2019/03/15 - 2019/04/11 毕业设计项目

## 环境配置

- JDK 1.8
- Tomcat 9.0
- MySQL 5.1
- Maven 3.5.4

## 系统架构

- Spring+SpringMVC+Mybatis

## 工具

- LogBack日志 MD5账户加密 DES数据库连接加密

## 第三方插件

- Echarts
